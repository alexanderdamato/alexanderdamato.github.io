Painting Canvas

This is an empty canvas, here you can paint whatever you want.

DIRECTIONS

You can use the mouse wheel to change the size of the brush,press the mouse to randomly change colors, and use the space bar to erase.

Use the right arrow to increase the red value and the Q key to decrease it.Use the left arrow to increase the blue value and the W key to decrease it.

Use the up arrow to increase the green value and the E key to decrease it.Use the down arrow to increase the alpha and the S key to decrease it.
